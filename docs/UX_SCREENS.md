# Silvertown Tunnel Inspection System - UX Screens

## Brand Guidelines

### INFRATEC Branding
- **Primary Color**: `#003366` (Deep Navy Blue)
- **Secondary Color**: `#FF6600` (Vibrant Orange - for accents/CTAs)
- **Success/Grade 1**: `#22C55E` (Green)
- **Warning/Grade 3**: `#F59E0B` (Amber)
- **Error/Grade 5**: `#EF4444` (Red)
- **Background**: `#F8FAFC` (Light Gray)
- **Card Background**: `#FFFFFF`
- **Text Primary**: `#1E293B`
- **Text Secondary**: `#64748B`

### Typography
- **Headings**: Inter (Bold)
- **Body**: Inter (Regular)
- **Monospace**: JetBrains Mono (for IDs, codes)

### Condition Grade Colors
```
Grade 1: #22C55E (Green)      - No deterioration
Grade 2: #84CC16 (Lime)       - Minor deterioration
Grade 3: #F59E0B (Amber)      - Moderate deterioration
Grade 4: #F97316 (Orange)     - Significant deterioration
Grade 5: #EF4444 (Red)        - Severe deterioration
```
